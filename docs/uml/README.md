# UML Diagrams

This folder contains PlantUML sources for the project architecture.

## Files

- `use_case_diagram.puml`
- `sequence_voice_session.puml`
- `sequence_multi_agent.puml`
- `sequence_admin_rag_upload.puml`
- `class_diagram.puml`

## Render commands

If PlantUML is installed:

```bash
plantuml docs/uml/*.puml
```

Or generate one file:

```bash
plantuml docs/uml/class_diagram.puml
```
