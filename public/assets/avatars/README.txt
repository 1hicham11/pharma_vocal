Place your 3D avatar files (.glb) here with these names:

- doctor.glb      -> for Medical / Doctor agents
- powerbi.glb     -> for Power BI specialist
- logistique.glb  -> for Logistics specialist
- sap.glb         -> for SAP specialist
- default.glb     -> fallback for other agents

The UI automatically tries to load these files.
If a file is missing, it falls back to emoji mascot.
